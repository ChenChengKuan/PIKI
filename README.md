# PKI

* Python 2.7up (it might not work for python 3)
* Apache Spark 1.5.1 up(If you want to run DesignerSimilarity.ipynb) 


Note: your should launch spark from command line by: IPYTHON_OPTS="notebook" bin/pyspark
